The forward pass

residual stream hook fires → grabs tensor active layer
Each layer reads the residual stream, adds its update, and writes it back. The hook is a clamp on the wire after that write (hook_resid_post): it sees the tensor [batch, seq, d_model] and lets it pass through unchanged.
